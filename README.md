# Student Management System

A Flask-based Student Management System built using Python, SQLite, and HTML.

## Features

- Add student records
- View all students
- Edit student details
- Delete student records
- SQLite database integration

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS

## Project Structure

- app.py - Flask application
- templates - HTML pages
- instance - Database storage
